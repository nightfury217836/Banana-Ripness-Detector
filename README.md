# How I Built a 🍌 Banana Ripeness Detector using Machine Learning | Computer Vision Project

<img width="1919" height="1042" alt="Screenshot 2026-02-28 225405" src="https://github.com/user-attachments/assets/877a76d1-7d71-47da-9fb3-b4a030f80408" />

In this project, I built an **AI-powered Banana Ripeness Detection System** using **Computer Vision** and **Deep Learning**.  
Using a modern object detection model (YOLO), the AI can analyze an image of a banana and classify its **ripeness stage in real time**.

This is a complete **End-to-End Computer Vision project** built with Python and a YOLO-based deep learning model.  
The AI learns to detect different banana conditions such as:

- Fresh Unripe
- Fresh Ripe
- Ripe
- Overripe
- Rotten
- Unripe  

by training on labeled images.

## 🚀 Project Workflow

Step-by-step, we will:

- ✅ Prepare and organize the dataset for training  
- ✅ Understand the dataset structure using `data.yaml`  
- ✅ Train a deep learning model using YOLO for object detection  
- ✅ Detect different banana ripeness stages automatically  
- ✅ Evaluate the model using **Precision, Recall, and mAP metrics**  
- ✅ Run predictions on new images to test the AI model
